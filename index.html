Projects

Predicting League of Legends

For the final project of my Introduction to Data Science, we had to analyze a data set. I love League of Legends, and I chose to focus my project around the game. I tried to predict the winner of a game using 180,000 matches using linear regression in RStudio. You can find the final project here.

Krane Multigames

Krane multigames is a CS:GO multigames map featuring 17 minigames, 8 of which are completely original. Each round, players have to jump around the center platforms to find the correct one. The winner gets to choose the minigame of the round. Minigames range from team battles to courses with a single winner. I don't update it as much as I used to, but it's one of my favorite projects. my profile.

Coop Retribution

Coop Retribution (working title) is a single-player experience built in CS:GO where I worked as the programmer in a 3-person team. CS:GO is a multiplayer game, and has limited functionality we could use for single-player mechanics. I set up an event-based system behind all of the events in the map, controlling player movement throughout the map and orchestrating setpieces. I created a spawning framework that allowed spawning new enemies to be very simple yet very customizable. I also created makeshift AI for a companion to provide fire support. The project isn't completed yet so it isn't public. Scripts were done in Squirrel.

Minecraft

My Minecraft multiplayer plugins use Bukkit/Spigot. My biggest and most recent project is the Dungeons plugin. The plugin allows servers to create timed courses for players to travel through. Players are able to compete and compare their fastest times. Data is handled in MongoDB, and the plugin is interfaced with using the command line. I've also released the SuggestionHelper plugin, allowing players to send suggestion tickets to the server, and a prototype for the SimplePoll plugin, which lets a server create polls.
