I don't update it as much as I used to, but it's one of my favorite projects.In the future I hope to create more CS:GO maps such as a de_ map based off of the de_krane minigame, or another minigame map based off the AWP ghosts minigame. HazeCraft

Currently, I am an administrator on the Minecraft survival server HazeCraft. Recently, the server has been acquired by the team at HostVenomLLC and Ready2Frag (which are both excellent server hosting companies that I wholeheartedly recommend, not just for Minecraft). Since the acquisition, we've been rebuilding the server into a revamped Minecraft network. I've been with the server for almost 5 years now, and it has an incredible community and staff team. I highly recommend you check it out!
